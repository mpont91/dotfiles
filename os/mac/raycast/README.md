## Raycast config

### Import / Export

Use Raycast app interface to import `raycast.rayconfig` file.

### Sortcuts

- `⌘ ⌃ ⌥ A` - Amazon Music
- `⌘ ⌃ ⌥ B` - Bitwarden
- `⌘ ⌃ ⌥ N` - Brackets
- `⌘ ⌃ ⌥ V` - Brave
- `⌘ ⌃ ⌥ D` - Discord
- `⌘ ⌃ ⌥ F` - Finder
- `⌘ ⌃ ⌥ G` - Google Chrome
- `⌘ ⌃ ⌥ I` - IntelliJ
- `⌘ ⌃ ⌥ O` - Microsoft Outlook
- `⌘ ⌃ ⌥ M` - Microsoft Teams
- `⌘ ⌃ ⌥ H` - Postman
- `⌘ ⌃ ⌥ P` - Preview
- `⌘ ⌃ ⌥ J` - RunJS
- `⌘ ⌃ ⌥ T` - iTerm
