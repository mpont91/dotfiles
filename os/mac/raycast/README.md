## Raycast config

### Import / Export

Use Raycast app interface to import `raycast.rayconfig` file.

### Sortcuts

- `⌘ ⌃ ⌥ D` - Discord
- `⌘ ⌃ ⌥ F` - Finder
- `⌘ ⌃ ⌥ G` - Google Chrome
- `⌘ ⌃ ⌥ I` - IntelliJ
- `⌘ ⌃ ⌥ O` - Microsoft Outlook
- `⌘ ⌃ ⌥ M` - Microsoft Teams
- `⌘ ⌃ ⌥ P` - Postman
- `⌘ ⌃ ⌥ T` - Warp
